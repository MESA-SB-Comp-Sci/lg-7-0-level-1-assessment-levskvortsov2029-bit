## Question 1 — Describe

What is the difference between a static unit like `px` and a responsive unit like `%` or `vh`?

Describe what makes a unit responsive and why that matters when building a website.

Static 'px' stays the same size no matter the device, while '%' or 'vh' changes the size, so it can fit different devices perfectly.
Change in size of the unit in different environments makes unit responsive.

## Question 2 — Explain

Look at these two CSS rules:

```css
.image {
  width: 400px;
}

.image {
  width: 50%;
}
```

Explain what happens to the image on a small screen with each rule. Why does one behave better than the other?

On a small screen with rule "width: 400px;" the image would be biger than the screen so you would only see the image very close as it will exeed the borders of screen, while with "width: 50%;" code, the image would change it's size to exactly 50% of the screen, so it will be visible.
